===============================================
PORTFOLIO — VALUES YOU NEED TO FILL IN
===============================================

Open src/data/portfolio.ts and replace these:

1. links.github
   Current: "NEEDS_LINK"
   Replace with: Your full GitHub profile URL
   Example: "https://github.com/yourusername"

2. links.linkedin
   Current: "NEEDS_LINK"
   Replace with: Your full LinkedIn profile URL
   Example: "https://linkedin.com/in/yourusername"

3. experience[1].date (Altab.ai)
   Current: "NEEDS_DATE"

4. projects[0].liveUrl (Conlang Dictionary)

5. projects[1].liveUrl (Multiplayer Game)
   Current: "NEEDS_LINK"
   Replace with: The deployed URL for the multiplayer game
   Or set to null if not currently deployed

6. projects[2].liveUrl (CubeSat at MSU)
   Current: "NEEDS_LINK"
   Replace with: The deployed URL for the CubeSat website
   Or set to null if not currently deployed

7. Resume PDF
   Place your resume at: public/resume.pdf
   The sidebar "View Résumé" button links to /resume.pdf

8. links.phone
   Current: "8324239132"
   If you do NOT want your phone number public, set to ""
   (The phone icon will not render if the string is empty)

9. About text
   Review the 3 paragraphs in portfolio.about
   Edit them to match your voice — these are drafts

10. Custom domain
    After deploying to Vercel, configure your custom domain
    in the Vercel dashboard under Settings → Domains
    Recommended: maximusyoung.dev or maximusyoung.com
